# えから便り（binsen）

描いた絵を、家庭用プリンタですぐ使える便箋やメッセージカードに整えるWebアプリです。
画像処理とPNG生成はすべてブラウザ内で行い、アップロード画像をサーバーへ送信しません。

## 主な機能

- JPG / PNG画像の選択・カメラ撮影
- 明るさ、紙の白さ補正、回転、拡大、位置の調整
- 透かし便箋、枠つき便箋、ちりばめ便箋、メッセージカード
- 枠つき便箋で選べる水彩・草花調のフレーム10種類
- 罫線の表示切り替えと短い印刷文字の入力
- A4縦・4面付け（300dpi相当）の印刷用PNG保存
- 1枚だけのPNG保存
- スマートフォン対応

## 起動方法

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:3000` を開きます。

## 使い方

1. 「絵を選ぶ・撮影する」から手書き絵の写真または画像を選びます。
2. 必要に応じて「絵を少し整える」から明るさ、白さ、向き、余白を調整します。
3. 4種類から使いたい便箋の形を選びます。
4. 完成プレビューを確認し、「印刷用A4を保存」を押します。
5. 保存したPNGをA4縦・実際のサイズ（100%）で印刷し、中央の点線で切ります。

## ファイル構成

```text
src/
  app/
    globals.css       紙ものUIとレスポンシブスタイル
    layout.tsx        メタデータと共通レイアウト
    page.tsx          トップページ
  components/
    letter-maker.tsx  操作UIと保存処理
    paper-preview.tsx A4完成プレビュー
  lib/
    canvas.ts         画像調整、テンプレート描画、PNG生成
```
